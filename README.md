# 🌸 Iris Flower Prediction App

A simple Machine Learning web application that predicts the species of an Iris flower based on its measurements.

---

## 📌 Features
- Predicts flower species:
  - Iris-setosa  
  - Iris-versicolor  
  - Iris-virginica  
- Takes user input for:
  - Sepal Length  
  - Sepal Width  
  - Petal Length  
  - Petal Width  
- Displays:
  - Predicted flower name  
  - Corresponding flower image  

---

## 🧠 Machine Learning
- Algorithm used: **K-Nearest Neighbors (KNN)**
- Data preprocessing: **StandardScaler**
- Dataset: **Iris Dataset**
